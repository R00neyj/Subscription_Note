- [x] **플랫 디자인(Flat Design)**: 앱 전반의 모든 그림자(Shadow) 제거 및 선(Border) 기반 미니멀 UI 구현
- [x] **온보딩 튜토리얼**: SVG Mask 기반의 스포트라이트 가이드 시스템 (기기별 동적 타겟팅 적용)

- [x] **데이터 시각화**: 카테고리별 지출 비중 바 차트 및 인터랙션 구현

### Phase 3: 백엔드 연동 및 보안
- [x] **Supabase 클라이언트 연동**: Cloud DB 연결 및 환경 변수(.env) 보안 설정
- [x] **구글 로그인 (Google Auth)**: Supabase 기반 소셜 로그인 및 사용자별 데이터 분리
- [x] **보안 정책 (RLS)**: 본인의 데이터만 접근 가능한 데이터베이스 보안 정책 적용

### Phase 4: 기능 고도화 및 품질 강화
- [x] **멀티 카테고리 시스템**: 추가/수정 모달 내 다중 카테고리 선택 및 필터링 로직 고도화
- [x] **전역 검색 시스템**: 헤더 검색창 연동 및 별도 검색 결과 페이지(`SearchResults.jsx`) 구현
- [x] **보안 및 유효성 검사**: 입력값 Sanitize(XSS 방지) 및 필수 항목 표시
- [x] **UI/UX 정교화**: 튜토리얼 위치 최적화, 대시보드 인터랙션 고도화
