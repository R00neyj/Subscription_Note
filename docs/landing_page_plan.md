# 구독노트(Sub-list Dashboard) 랜딩 페이지 구축 계획

## 1. 핵심 컨셉: "잊고 있던 나의 가능성을 깨우는 파트너"
딱딱한 자산 관리를 넘어, 사용자가 무심코 흘려보낸 구독료의 가치를 발견하고 더 나은 일상을 설계할 수 있도록 돕는 따뜻하고 지능적인 동반자임을 강조함.

## 2. 페이지 구조 (Storytelling Flow)

### 2.1. Hero Section: 공감과 발견
- **카피**: "혹시 오늘도 보지 않는 영상에 소중한 하루치를 지불하셨나요?"
- **비주얼 (키비주얼 및 애니메이션 상세)**:
  - **초기 상태 (Floating)**: 화면 곳곳에 사용자가 흔히 쓰는 구독 서비스 로고가 새겨진 작은 동전/버블들이 천천히 둥둥 떠다님 (자연스러운 Float Animation).
  - **인터랙션 트리거 (Gathering)**: 페이지 진입 후 1~2초 뒤, 혹은 사용자가 스크롤을 시작하면, 흩어져 있던 동전/버블들이 화면 중앙으로 부드럽게 모여듦 (Natural Easing).
  - **변환 (Transformation)**: 동전들이 하나로 뭉쳐지면서 기분 좋은 반짝임(Particle Effect)과 함께 사용자가 평소 원했던 '실제 보상'(예: 커피 머신, 비행기 티켓 일러스트 등)으로 형태가 변함 (가벼운 탄력의 Spring Animation).
  - **의도**: 구독 취소가 '박탈'이 아닌 '나를 위한 선물'이 된다는 메시지를 직관적이고 다정하게 전달.
- **가치 제안**: 숨겨진 지출을 찾아내어 당신의 꿈을 위한 자본으로 바꿔주는 경험.

### 2.2. Value 1: 일상이 바뀌는 숫자 (Real-life Reward)
- **메시지**: "구독료만 모아도 내년엔 맥북으로 작업을 시작할 수 있어요."
- **설명**: 1년, 5년 뒤의 누적 금액을 단순히 숫자로 보여주지 않고, 사용자가 평소 갖고 싶어 했던 '실질적인 선물'로 치환하여 시각화. "당신의 구독료는 이미 맥북 한 대만큼의 가치가 있습니다."

### 2.3. Value 2: 세심한 배려 (Gentle Guide)
- **메시지**: "잊고 계셨죠? 이 구독은 지난달 한 번도 사용하지 않았네요."
- **기능 강조**:
  - **다정한 조언**: "이 서비스는 어떠신가요? 요즘 만족도가 낮아 보여요."와 같은 대화형 알림.
  - **현명한 선택**: 비슷한 서비스들 사이에서 고민할 때, 사용자의 패턴을 분석해 가장 잘 맞는 하나를 추천해 주는 가이드.

### 2.4. Value 3: 심리적 안정감 (Peace of Mind)
- **메시지**: "결제일마다 가슴 철렁할 일, 이제는 없도록."
- **기능 강조**: 캘린더를 통해 다가올 결제일을 미리 다정하게 알려주고, 예산 안에서 안전하게 구독을 유지할 수 있도록 돕는 스마트 알림.

### 2.5. Experience: 사람을 향한 디자인
- **메시지**: "쓰면 쓸수록 기분 좋아지는, 당신만을 위한 공간"
- **포인트**: 
  - 기계적인 데이터 나열이 아닌, 사용자의 성장을 응원하는 메시지 배치.
  - 눈이 편안한 색감과 손끝에 감기는 부드러운 반응형 인터랙션.
  - 모바일에서도 마치 대화하듯 편안하게 조작할 수 있는 직관적인 UI.

### 2.6. Final CTA: 함께 시작하기
- 서비스 진입 버튼 및 "지금 바로, 당신의 더 나은 내일을 위해 구독 다이어트를 시작해 볼까요?" 문구.

## 3. 시각적 가이드라인 (Visual Tone & Manner)
- **Color**: 깊이감 있는 어두운 톤에 따뜻한 포인트 컬러를 사용하여 '전문적이면서도 친근한' 느낌 전달.
- **Motion**: 기계적인 전환이 아닌, 생명력이 느껴지는 유기적이고 탄력 있는 움직임(Natural Easing).
- **Typography**: 숫자는 신뢰감 있게, 문구는 마치 옆에서 말을 건네는 듯한 부드럽고 친절한 서체 사용.

## 4. 기대 효과
- **브랜딩**: '관리 도구'를 넘어 사용자의 경제적 자유를 응원하는 '퍼스널 코치'로 각인.
- **전환율**: 사용자 개개인의 삶에 밀착된 비유와 공감형 문구를 통해 서비스 가입에 대한 심리적 허들을 낮춤.
