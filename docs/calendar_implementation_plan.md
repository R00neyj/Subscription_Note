# 캘린더 뷰 구현 계획 (MVP) [완료]

**마지막 업데이트**: 2026. 02. 02
**목표**: 가장 간단하고 직관적인 방법으로 월별 구독 결제일을 시각화합니다.
**전략**: 무거운 캘린더 라이브러리 대신, `date-fns`와 `CSS Grid`를 활용한 커스텀 구현으로 디자인 일관성과 성능을 확보합니다.

---

## 1단계: 필수 라이브러리 설치 및 유틸리티 구성 [DONE]
- **라이브러리**: `date-fns` 설치 완료.
- **데이터 정제**: `src/lib/dateUtils.js`에 날짜 파싱 및 확인 로직 구현 완료.

## 2단계: 캘린더 레이아웃 구조 잡기 (`Calendar.jsx`) [DONE]
- **Header**: 년/월 표시 및 이동 버튼, '오늘' 버튼 구현 완료.
- **Week Days**: 일~토 헤더 (주말 색상 차별화) 구현 완료.
- **Days Grid**: 7열 Grid 기반 유연한 레이아웃 구현 완료.

## 3단계: 구독 데이터 매핑 및 시각화 [DONE]
- **로직**: Zustand 스토어 연동 및 각 날짜별 구독 필터링 완료.
- **UI 표현**: 
  - 카테고리별 컬러 점(Dot) 표시.
  - PC 환경에서 서비스명 텍스트 표시.
  - 항목 초과 시 `+N` 축약 표시 및 오늘 날짜 하이라이트 구현 완료.

## 4단계: 하단 리스트 뷰 (Selected Date Info) [DONE]
- **인터랙션**: 날짜 선택 시 해당 날짜의 결제 항목 상세 리스트 노출 완료.
- **정보**: 서비스명, 금액, 결제 수단, 카테고리 및 해당 일자 총 합계 금액 표시 완료.
- **연동**: 리스트 항목 클릭 시 수정 모달 연결 완료.

---

## 구현 순서
1. `npm install date-fns` 명령어 실행.
2. `src/utils/dateUtils.js` (또는 내부 함수) 생성: 날짜 파싱 로직.
3. `Calendar.jsx` UI 스켈레톤 작성 (Header + Grid).
4. 데이터 매핑 및 스타일링 (다크 모드 대응).
